# GuildOS

GuildOS is a World of Warcraft Retail guild management addon with a modern UI.

## Features
- Guild roster overview with filters
- Guild and officer chat history
- Lightweight addon sync peer discovery (PING/PONG)
- Local activity timeline

## Install
1. Copy this folder as `GuildOS` into `_retail_/Interface/AddOns/`.
2. Ensure `GuildOS.toc` is present at the addon root.
3. Reload UI with `/reload`.

## Slash commands
- `/gos` open GuildOS
- `/gos default` open Blizzard default guild/communities panel
- `/gos replace on|off` toggle replacement behavior
- `/gos sync` send sync ping
